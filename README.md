# TEMPLATE API
Template API with user login, access token ect, docker-compose.yml to build db instance

## Supported features
- [x] Mongodb
- [x] Express
- [x] Swagger
- [x] Jest
- [x] Multer

## Test
```
yarn test
```

## Live Dev
```
yarn dev
```

## Build
```
yarn build
```

## Start
```
yarn start
```
